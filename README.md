# Intern Git Practice Lab

This repository is used for practicing Git workflows during the internship program.

Each group or individual must work on its own branch, make commits, push changes, and open a Pull Request.

## Project Purpose

The purpose of this repository is to help interns practice common Git tasks such as creating branches, tracking changes, committing updates, pushing work to GitHub, and opening Pull Requests.

This project also introduces a simple robotics-related structure so interns can practice editing files in a realistic development workflow.

The project helps interns learn how to:
- Create and manage branches
- Track file changes using Git
- Commit changes with meaningful messages
- Push work to GitHub
- Create Pull Requests
- Collaborate using standard software development workflows
