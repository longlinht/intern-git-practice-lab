# Robot Startup Checklist

- Verify that the robot battery level is enough before starting any test. If the battery is low, charge or replace it before running the robot.
- Check the network connection between the laptop and the robot. If the robot is connected using Ethernet, make sure the laptop IP address is manually set to the correct static IP range. If the robot is connected through a router, make sure both the laptop and the robot are connected to the same network.
- Confirm that the laptop can communicate with the robot by using a ping test before starting any command.
- Check that the emergency stop button is working and easy to reach before the robot starts moving.
- Make sure the robot is placed in a safe and clear area with enough space around it.
- Check that the robot is in the correct starting mode before testing. Do not move directly to walking mode without preparing the robot first.
- Make sure all cables are connected properly and are not blocking the robot movement.
- Keep people away from the robot during startup, especially when the robot is changing mode or starting motion.
- Stop the test immediately if the robot moves unexpectedly, loses balance, or goes outside the safe testing area.
