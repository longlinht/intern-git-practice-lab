# Robot Startup Checklist

- Clear a 3 m × 3 m workspace
- Charge the hand controller fully
- Inspect all joints manually
- Lock the battery pack securely
- Check the torso power indicator
- Hold power button for 3–5 seconds
- Confirm the emergency stop functional status
- Bind the remote control connection
- Verify joints stiffen and stand upright (RT+Y)
- Ping the default 192.168.10.10 IP address
