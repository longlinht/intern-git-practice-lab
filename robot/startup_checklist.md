# Robot Startup Checklist

- Verify the robot battery level is within the approved operating range.
- Confirm the emergency stop is accessible, released, and functioning correctly.
- Check that the robot is connected to the expected network.
- Confirm the operator workstation can communicate with the robot control software.
- Inspect sensors and visible cabling for loose connections or damage.
- Verify the latest required configuration files are loaded before enabling motion.
- Start with the robot in a clear test area before running any movement commands.
