# Robot Startup Checklist

- Verify the robot battery level is sufficient and within the approved operating range.
- Check network connection and confirm the robot is reachable from the operator station.
- Confirm the emergency stop is accessible, released, and functioning correctly.
- Confirm the robot boots correctly without startup errors.
- Verify sensors are detected and reporting expected status.
- Confirm required ROS nodes and services are ready if applicable.
- Verify the latest required configuration files are loaded before enabling motion.
- Inspect the workspace for obstacles before starting movement tests.
